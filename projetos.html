Projetos GitHub

Todos os projetos são para fins de aprendizado e podem haver bugs ou até não funcionarem.

Timeday-Website:

Este site é um exercício de aprendizado de javascript. Mostra as horas do dia de acordo com o relógio. Há também uma imagem dinâmica representando o tempo. Exercício do Curso em Vídeo.

Age-Checker

Esse site tem objetivo de treinar validação de dados, o código identifica a idade representada no input e exibe um imagem validando a informação. Exercício do Curso em Vídeo.

Multiplication-Table-Website

Esse site calcula e exibe uma tabuada de qualquer número informado no input, exercício simples de coleta, validação e injeção de dados. Exercício do Curso em Vídeo.

HTML-DEV-FORMS

Front-end de um formulário básico, exercício de coleta de dados e CSS. Exercício de Rafa Ballerini 🌐 https://youtu.be/wwqOJ2o84S4

Counter-EX

Esse site simula um contador com passo, exercício que coleta, validação de dados. Exercício do Curso em Vídeo.

Dynamic-memories

Esse site é um presente de natal para minha namorada 🥰 Ele é um espaço para guardar nossas memórias em um lugar só.
